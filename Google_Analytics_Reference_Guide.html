üåªCustom Parameters by Event Type

üåºquestion_answered Event

Tracks when a user answers any question in the quiz

question_number Which question (1-8)
question_id The question's ID
answer_text The text of the answer chosen (truncated to 100 chars)
is_final_question Whether this was the last question (true/false)

üíïquiz_completed Event

Tracks when a user finishes the entire quiz

planet Selected planet (e.g., "Visualis", "Textara", "Codion")
creature Selected creature (e.g., "Nebula Neko", "Speech Sprite")
top_tool Name of the #1 recommended tool
top_tool_score Score of the #1 recommended tool
total_questions How many questions were answered
total_tools_recommended How many tools were recommended (should be 10)

üê±tool_clicked Event

Tracks when a user clicks on a tool link

tool_name Name of the tool that was clicked
tool_url URL of the tool
tool_rank Position in recommendations (1 or 2)
compatibility_percentage Cosmic compatibility percentage shown
tool_category Category of the tool
tool_price Price type (free, freemium, paid)

ü¶ã Pro Tip: Start by registering the 4 High Priority dimensions first. Once you have data flowing and start analyzing it, you can always add more dimensions based on what insights you want to dig deeper into!

üå∫How to Register Custom Dimensions

  1. Go to Admin (gear icon in bottom left of Google Analytics)
  2. In the Property column, click Custom definitions
  3. Click Create custom dimension
  4. For each parameter:
    • Dimension name: Use a friendly name (e.g., "Planet")
    • Scope: Select "Event"
    • Description: Add a brief description
    • Event parameter: Enter the exact parameter name (e.g., "planet")
  5. Click Save
  6. Repeat for each parameter you want to track
üíê Important: Custom dimensions can take 24-48 hours to start collecting data. Events sent BEFORE you create the dimension won't be retroactively updated, but all new events will include the dimension automatically.
